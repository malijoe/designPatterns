What is WebexKubed?

WebexKubed is a platform built on top of Kubernetes and cloud native technologies to provide a modern and resilient container platform for application workloads across the Collaboration Business Unit. It's a multi-provider solution — built with end-to-end automation, scalability and security at its core; developed by the Webex Cloud Native Platform Engineering team to provide state-of-the-art Platform as a Service (PaaS) for Webex.

Cost efficient

Cost and performance efficiency at the core with features such as autoscaling and elastic infrastructure.

Built for Webex developers

Modular design and APIs allows application groups to enhance and extend the platform – driving innovation.

Cloud agnostic

Enable uniform delivery of applications across multiple cloud providers.

Enable re-use

Common platform layer across calling, meetings and messaging enables reuse and accelerates feature development.

High Availability

Resilient to failure, with automatic workload rescheduling.

Security first

In-built security; reduces the security burden on application groups whilst maintaining software hygiene.
